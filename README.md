# Excel Admin Automation Suite

A collection of high-performance VBA utilities designed to streamline office administration and file management.

## Tools Included

### 1. Document Folder Automator
Automatically creates a structured folder hierarchy based on Excel data. Perfect for onboarding new employees or setting up project folders.
- **Features**: Custom root directory selection and multi-level sub-folder creation.

### 2. File Renamer & Archive Tool
Batch rename files based on Excel mapping to clean up messy downloads or standardized archives.
- **Features**: Direct renaming and status logging.

## Installation
1. Download the `Admin_Automation_Template.xlsm` (coming soon) or use the `.bas` files.
2. To use the `.bas` files:
   - Open Excel and press `Alt + F11`.
   - Go to `File > Import File...` and select the files from the `/src` folder.

## How to Use
- **Folder Automator**: Enter your target directory in the settings cell and list your folder names in Column A.
- **File Renamer**: List the current filename in Column A and the new filename in Column B.
